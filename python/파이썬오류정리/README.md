## 1) [에러명: 예시] Gemini API 토큰 소모량 초과 에러
- **문제 발생:** 대량의 뉴스 데이터를 한 번에 분석할 때 토큰 제한(Rate Limit)으로 시스템 중단
- **원인 분석:** 분석할 텍스트의 길이가 모델의 최대 입력 범위를 초과함
- **해결 방법:** 1. 데이터 분할 처리(Chunking) 로직 구현
    2. API 호출 사이에 지연 시간(Sleep) 추가
- **결과:** 시스템 안정성 100% 확보 및 예외 처리 완료

[👉 해결 코드 확인](./api_timeout_fix.py)
